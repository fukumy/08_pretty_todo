# ベルサイユの薫り TODOリスト (My Royal Todo)

「ベルサイユの薔薇」の世界観を彷彿とさせる、華美でエレガントな乙女チックTODOリストアプリケーションです。

![Main View](/C:/Users/mika1/.gemini/antigravity/brain/3f8ef341-efa1-48e9-ae0d-e61404bf79b5/check_horizontal_scroll_1774043594765.png)

## 🌹 特徴
- **ロイヤルなデザイン**: 金色のフレーム、薔薇の装飾、気品溢れる配色（ロイヤルレッド＆ゴールド）。
- **優雅な演出**: タスク完了時に「きらめき（✨）」が舞う、ドラマチックなエフェクト。
- **オリジナルアイコン**: AIによって生成された、深紅の薔薇と金細工のロイヤルアイコン。
- **PWA対応**: iPhoneやAndroidのホーム画面に追加して、本物のアプリのように起動可能。
- **レスポンシブ設計**: PCから超小型スマホ（280px）まで、あらゆるデバイスで美しく表示。

## ✨ 使い方
1.  **タスクの追加**: 入力欄に麗しき日々の予定を記し、「追加」ボタンを押します。
2.  **完了の祝杯**: 完了したタスクをタップすると、きらめきと共にリストから消え去ります（あるいは線が引かれます）。
3.  **ホーム画面への追加**: ブラウザのメニューから「ホーム画面に追加」を選択することで、いつでもお手元にベルサイユの薫りを。

## 🛠️ 技術スタック
- **Frontend**: HTML5, CSS3 (Vanilla JS), JavaScript (ES6+)
- **Design**: Google Fonts (`Dancing Script`, `Playfair Display`, `Noto Serif JP`)
- **Graphics**: AI-generated Royal Icon (DALL-E inspired)
- **Deployment**: GitHub Pages
